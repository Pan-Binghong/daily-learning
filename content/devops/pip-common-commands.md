Pip是 Python 的包管理工具，全称为 "Pip Installs Packages"，也可以理解为 "Pip Installs Python"。它用于安装和管理 Python 的第三方库和模块，类似于其他语言中的包管理工具，比如 JavaScript 的 npm 或 Ruby 的 gem。
